A simple and interactive XO (Tic Tac Toe) game built using HTML, CSS, and JavaScript.
This project focuses on DOM manipulation, game logic, and clean UI design.

🚀 Features

🎮 Two-player gameplay (X vs O)

⚡ Real-time move updates

🧠 Win & draw detection logic

🔄 Restart game functionality

💻 Responsive and clean UI

🛠️ Tech Stack

HTML5

CSS3

JavaScript (Vanilla JS)
